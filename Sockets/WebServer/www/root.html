You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html

File Structure in www (you can use /file/www/FILENAME):

${links}

Additional Requests

Multiply

Multiply 2 Integers

Example: /multiply?num1=5&num2=3

Expected Output: Result is: 15

GitHub

Displays full_name, ID, and login of the owner of the public repos.

Example: /github?query=users/amehlhase316/repos

Conference Championship Winner

Retrieve the winner of the AFC or NFC championship for a specific year*

*Between 2020 & 2025

Example: /conferencewinner?year=2020&conference=AFC

Expected Response: Year: 2020 , Conference: AFC , Winner: Kansas City Chiefs

Anagram Checker

Check if two words are anagrams of each other.

Example: /anagram?word1=listen&word2=silent

Expected Response: word1 = listen, word2 = silent, Are Anagrams?: Yes
